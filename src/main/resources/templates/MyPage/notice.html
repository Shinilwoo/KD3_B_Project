  • [공지] [재공지] 이메일 발신자 주소 등록 유의사항 (수정 전: 이메일 발신자 주소 등록 해제 및 기본 주소 제공 안내) [ ▲ ]

    Date: 2024.03.19

    이벤트 테크 플랫폼, 이벤터스를 이용해 주셔서 감사합니다. 플랫폼에서 제공하고 있는 이메일 기능 관련하여 주요 변경사항을 공지드리오니, 서비스 이용에 참고 부탁드리겠습니다. 2024년 02월부로 적용된 Gmail의 운영 정책에 따라 이메일 서비스 제공 업체(이벤터스에 해당)를 사용하여 이메일을 발송하는 경우, -@gmail.com 주소의 수신자들에게 정상적인 발송이 제한되고 있습니다. 모든 참가자에게 호스트의 안내 이메일을 성공적으로 전달하는 것을 최우선으로 고려하여, 이벤터스에서는 인증된(발송이 제한되지 않는) 발신자 이메일 주소를 기본 발신자 주소로 제공합니다. 호스트는 직접 발신자 이메일 주소를 등록 후 사용할 수 있으나, -@gmail.com 이메일 주소를 사용하는 참가자들에게는 발송에 실패할 확률이 높습니다. [변경사항 요약] 1차 - 변경 내용: 이메일 발신자 주소를 일괄 기본 주소로 변경(no-reply@event-us.kr) - 변경 일시: 2024년 3월 5일 오전 10시 2차 - 변경 내용: 이메일 발송 시 기본 발신자 주소를 제공, 변경 가능(선택 사항) - 변경 일시: 2024년 3월 19일 오전 10시 [참고사항] *3월 19일(화)부터 발신자 주소 신규 등록이 가능합니다. 단, 발신자 주소를 등록하더라도 gmail을 사용하는 수신자에게는 발송이 불가하오니 착오 없으시길 바랍니다. *개별 발신자 주소로 발송하길 원하시는 경우, 참가자리스트 다운로드 후 별도로 발송 가능합니다. *Gmail 운영정책 확인(클릭) 호스트 여러분의 너른 양해를 부탁드리며, 문의 또는 불편사항은 채팅상담을 통해 접수해 주시면 감사하겠습니다. 감사합니다.

  • [공지]페이스북 간편 로그인 제공 종료 안내 [ ▲ ]

    Date: 2024.02.06

    이벤트 테크 플랫폼, 이벤터스를 이용해 주셔서 감사합니다. 회원가입, 로그인 옵션으로 제공하고 있는 페이스북 간편 로그인 기능을 2024년 3월 5일부로 제공 종료합니다. 페이스북 간편 로그인 관련 지속적인 문의를 접수하여 고민한 결과, 서비스 개선의 일환으로 페이스북 간편 로그인 계정을 일반 계정으로 일괄 전환하고자 합니다. 서비스를 계속해서 이용하시기 위해서는 새로운 비밀번호를 설정해 주셔야 합니다. 해당 회원 분들은 아래 내용을 참고하시어 기간 내 비밀번호 설정을 완료해 주시길 바랍니다. [기능 종료 요약] - 종료 기능: 페이스북 간편 로그인 - 종료 일시: 2024년 3월 5일 오전 10시 - 변경사항: 페이스북 간편 로그인 이용 불가, 일반 로그인 비밀번호 설정 후 이용 가능 [일반 로그인 비밀번호 설정 안내] - 비밀번호 설정 기간: 2024년 2월 6일 오전 10시 ~ 2024년 3월 5일 오전 9시 - 비밀번호 설정 방법: 페이스북 간편 로그인 → 비밀번호 설정 화면으로 자동 이동 → 비밀번호 설정 → 일반 로그인 이용 - 문의: 채팅상담 (평일 오전 10시 ~ 오후 5시) [참고사항] - 일반 로그인 계정으로 전환하더라도 기존의 이용 내역은 그대로 유지됩니다. - 구글, 네이버, 카카오 계정 로그인은 계속해서 제공됩니다. 단, 일반 로그인 계정에 다른 간편 로그인을 설정하는 것은 불가합니다. - 기간 내 비밀번호 설정을 완료하지 않은 경우, 로그인 페이지의 ‘비밀번호 찾기’를 이용해 주시길 바랍니다. 회원 여러분의 너른 양해를 부탁 드리며, 저희 이벤터스는 더욱 안정적으로 서비스를 제공하기 위해 최선을 다하겠습니다. 문의사항은 언제든 채팅상담을 이용해 주시길 바랍니다. 감사합니다.
